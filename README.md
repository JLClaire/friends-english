📚 Clip Trainer

一个基于影视片段的英语影子跟读与主动输出训练工具。
通过结构化的 4 步学习流程，帮助你真正提升听力理解、语音模仿与英文表达能力。

本项目采用纯前端实现，可直接部署在 GitHub Pages，无需服务器。

🎯 项目定位

很多人学英语时会出现这些问题：

依赖中文字幕，理解很快但听力进步很慢

看得懂台词，却说不出来

听力输入多，但主动输出很少

Clip Trainer 的核心目标是：

让你“先听、再读、再确认、再输出”，
从被动理解转向主动表达。

🚀 核心功能
🎬 四步学习流程
Step 1 — 无字幕观看

直接观看片段

训练整体听感

不依赖文本

Step 2 — 英文 Shadowing 模式（影子跟读）

默认进入 单句模式（Single View）

支持功能：

点击句子 → 自动跳转播放 → 到句尾自动暂停

左右箭头（或键盘 ← / →）切换句子

倍速播放调节

中文默认隐藏

可手动显示 / 隐藏中文辅助理解

训练逻辑：

先听英文

跟读模仿语音语调

听不懂再显示中文确认

再次隐藏中文重复跟读

目的：避免“中文抢先理解”，优先训练听力与输出。

Step 3 — 中译英输出训练

显示中文字幕

你自行写出英文表达

自动本地保存（localStorage）

强制进行主动表达

这是从“模仿”到“表达”的关键阶段。

Step 4 — 对照与总结

查看原句对照

对比自己的表达

双击可跳转播放对应句子

🧠 设计理念

本项目基于三个学习原则：

1️⃣ 听力优先

语言首先是声音，而不是文字。

2️⃣ 英文优先

中文用于确认理解，而不是替代理解。

3️⃣ 输出巩固记忆

主动写出来，比反复阅读更有效。

📦 字幕格式说明

每个字幕文件为 JSON 格式：

[
  {
    "start": 1.70,
    "end": 3.10,
    "en": "Alright, so I'm back in high school.",
    "zh": "好吧，我又回到了高中。"
  }
]
➕ 如何添加新视频

将视频文件放入：

/assets/videos/

将字幕 JSON 文件放入：

/data/

在 data/clips.json 中新增配置：

{
  "id": "clip_id",
  "title": "片段标题",
  "videoSrc": "./assets/videos/视频文件名.mp4",
  "linesSrc": "./data/字幕文件名.json"
}

无需修改 JS。
